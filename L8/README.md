# Лабораторная работа №8  

Изучение jQuery: подсветка поста и эффект для логотипа.

## Что реализовано

- подключение jQuery
- подсветка поста при наведении (hover)
- плавная анимация логотипа (css, transform)
- структура проекта для GitHub

## Файлы

- index.html — страница
- style.css — стили оформления
- script.js — эффекты jQuery

## Как запустить

Открыть файл index.html в браузере.

## Требования

аналогично Л.Р. 7
