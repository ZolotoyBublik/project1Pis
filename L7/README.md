# Лабораторная работа №7  

Основы JavaScript и работа с DOM.

## Что реализовано

- простейшие функции JavaScript
- работа с параметрами
- условные операторы
- массивы и циклы
- изменение DOM‑элементов
- обработка событий (клик по кнопке)

## Файлы

- index.html — запуск базовых JS‑функций
- script.js — функции, операторы, массивы
- dom.html — страница для работы с DOM
- dom.js — изменение текста и стилей через JS

## Как запускать

Открыть файлы в браузере:

- index.html — посмотреть вывод в консоли
- dom.html — протестировать изменение DOM

## Требования

Браузер с поддержкой JavaScript (Chromium и производные, Firefox, Edge, UC, Vivaldi и проч.).
